# A1 – Injection（注入攻擊）

>*注入攻擊漏洞，例如SQL，OS以及 LDAP注入。這些攻擊發生在當不可信的資料作為命令或者查詢語句的一部分，被發送給解譯器的時候。

>*攻擊者發送的惡意資料可以欺騙解譯器，以執行計畫外的命令或者在未被恰當授權時訪問資料。
